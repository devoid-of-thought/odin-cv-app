# Odin CV Maker

A CV Builder created with React Js. 

Made as an assignement for [Project Odin](https://www.theodinproject.com/lessons/node-path-react-new-cv-application) 

**Live Project Demo:** Deployed with [netlify](https://devoid-of-cv.netlify.app).

## Usage

Input your name, contact details and email to create header.

Add and edit educational or work qualifications.
